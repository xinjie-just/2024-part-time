# CSS 命名推荐

CSS 命名是一个繁琐又细致的活儿，担心可能重复，结果写出了超长的类名。下面将常见的功能型命名总结如下：

## 1. 状态

```
前一个    prev
后一个    next
当前的    current

显示的    show
隐藏的    hide
打开的    open
关闭的    close

选中的    selected
有效的/激活的    active
默认的    default
反转的    toggle

禁用的    disabled
危险的    danger
主要的    primary
成功的    success
提醒的    info
警告的    warning
出错的    error

超大的    xl
大型的    lg
中型的    md
小型的    sm
超小的    xs
```

## 2. 布局

```
文档    doc
头部    header
主体    body
尾部    footer
主栏    main
侧栏    side
容器    box/container/wrapper
```
